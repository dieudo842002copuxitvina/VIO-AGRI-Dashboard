🔧 CORE MODULES

Decision Engine
  Purpose: Analyze market conditions & generate insights
  Features: 5 business rules, caching, timeout protection, confidence scoring
  Input: User profile + market data
  Output: Insights array + recommendations + summary
  Performance: 50-150ms (cold), <10ms (cached)
  Status: ✅ Production-ready

Recommendation Engine
  Purpose: Generate personalized product & action recommendations
  Features: Multi-factor scoring (interest, market, climate, quality)
  Scoring: Interest 30% | Market 35% | Climate 25% | Quality 10%
  Products: 10+ in catalog (IoT, fertilizer, equipment)
  Output: Ranked recommendations with confidence & reasons
  Performance: 200-400ms
  Status: ✅ Production-ready

React Components
  useDecision Hook:       State management for Decision API (5-min cache)
  InsightFeed:           Display market insights (categories, badges)
  RecommendationPanel:   Display recommendations (priority-based colors)
  Plus 7 UI components:  Sidebar, Header, PriceChart, Weather, etc.
  Status: ✅ All working
